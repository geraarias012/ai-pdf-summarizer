# AI PDF Summarizer

Sistema que utiliza Inteligencia Artificial para analizar y resumir documentos PDF automáticamente.

## Features

- extracción de texto de PDFs
- generación automática de resúmenes
- interfaz simple
- posibilidad de extender a chatbot sobre documentos

## Tecnologías

- Python
- NLP
- LLM
- Streamlit

## Instalación

1. Clonar repositorio
2. Crear entorno virtual
3. Instalar dependencias

## Uso

Subir un PDF y generar resumen automáticamente.
